# Ingest

Use this when the user gives material that should become durable context.

## Routing

- Article or pasted web text: `article-enrichment`
- Idea or link: `idea-ingest`
- PDF, image, audio, video, transcript, or repo media: `media-ingest`
- Voice note or dictated thought: `voice-note-ingest`
- Book or long PDF: `book-mirror` or `strategic-reading`

Keep source text and exact user wording when it matters.
