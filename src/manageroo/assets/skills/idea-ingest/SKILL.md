# Idea Ingest

Capture reusable ideas without making a giant report.

## Output shape

- title
- source
- exact user wording when provided
- short summary
- why it matters
- possible MANAGEROO lane: memory, document, skill, installer, docs, or backlog

Do not overbuild. File the idea so it can be found later.
