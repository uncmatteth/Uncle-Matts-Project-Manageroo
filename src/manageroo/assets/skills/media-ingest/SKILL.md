# Media Ingest

Use this when the source is not just normal text.

## Rules

1. Preserve the original file path or source link.
2. Extract text when a local tool can do it.
3. Record media metadata and what was not understood.
4. Do not pretend OCR, PDF text extraction, or thumbnails equal real vision.
5. If visual understanding matters, ask for or configure a real media evidence lane.
