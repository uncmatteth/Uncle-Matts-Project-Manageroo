# Citation Fixer

Use this before publishing or relying on research notes.

## Rules

1. Every factual claim needs a source.
2. User statements should be marked as user statements.
3. Web claims need link and date.
4. Do not invent citations.
5. If a source cannot be verified, mark the claim as unverified.
