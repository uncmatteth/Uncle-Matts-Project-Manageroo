# Book Mirror

Use this for books and very long documents.

## Rules

1. Work chapter by chapter.
2. Do not read ahead when the task only needs the current slice.
3. Preserve chapter order and source anchors.
4. Produce a brain page first.
5. PDF export is optional through `brain-pdf`.

This is not a normal summary lane. It is a personalized long-document lane.
