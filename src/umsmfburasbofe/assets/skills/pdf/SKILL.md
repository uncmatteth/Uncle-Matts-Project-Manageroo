# PDF

Use this for PDF-specific work.

## Rules

1. Extract text when possible.
2. Inspect page count and layout when rendering matters.
3. Treat missing fonts, broken glyphs, and frontmatter leaks as real defects.
4. Do not call a PDF good without checking the produced file.
5. For brain exports, brain page first, PDF second.
