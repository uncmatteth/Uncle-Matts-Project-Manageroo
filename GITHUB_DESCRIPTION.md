# GitHub Description Copy

## Repository Description

```text
A local command-line tool that helps AI coding agents build or repair a Git repo from one plain-English brief, with checks, review, and evidence.
```

## Plain-English About Text

- UMSMFBURASBOFE is the acronym, the command, and the public name.
- It is for people who use AI coding agents on real Git repos.
- You write one plain-English brief saying what should be built or fixed.
- It reads the repo, makes a plan, gives the agent smaller jobs, runs checks, reviews the result, and saves a report.
- It exists because giant AI chats lose scope. The tool keeps the work tied to files, checks, review notes, and evidence.
- It was built around GBrain, GitNexus, Obsidian, AUTOREVIEW, Clawpatch, and any AI IDE or CLI agent that can run commands in the repo.
- It does not require one special version for Codex, Claude Code, Gemini, Grok, or another AI tool. They can all use the same installed command if they can work in the repo.
- It is alpha software.
- The first real run should be on a clone, branch, or disposable copy.
- It is not a replacement for CI, backups, security review, or human product judgment.
