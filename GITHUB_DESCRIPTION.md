## Repository Description

```text
A local controller for agentic builds: one brief into stack-aware context, scoped agent work, checks, review, repair, and evidence.
```

## Plain-English About Text

- UMSMFBURASBOFE is the acronym, the command, and the public name.
- It helps a configured agent stack work on a real Git repo without treating the whole project like one giant chat.
- You write a product brief in normal language.
- It maps the repo, pulls bounded context from tools like GBrain and GitNexus when configured, gives an agent bounded tasks, runs checks, reviews the result, and writes a final report.
- It was built for the local stack around GBrain, GitNexus, Obsidian, AUTOREVIEW, Clawpatch, and whatever AI IDE or CLI agent can run the controller.
- It is alpha software.
- The first real run should be on a clone, branch, or disposable copy.
- It is not a replacement for CI, backups, security review, or human product judgment.
